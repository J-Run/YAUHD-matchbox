# YAUHD-matchbox

Yet Another Ultimate Hacking Device 

Cheap DIY network tool based on OrangePi R1

* Fully autonomous for 24+ hrs
* x2 Ethernet
* onboard Wi-Fi
* x2 USB for extra peripherals (eg 4g / wifi / CrazyRadio PA etc)
